## Команды сохранения

*Сохраняет данные, добавленные с помощью git add*

```sh
git commit
```

*Команда, позволяющая откатить сохранения, созданные с помощью git commit*

```sh
git reset
```

*Позволяет посмотреть историю коммитов*

```sh
git log
```
